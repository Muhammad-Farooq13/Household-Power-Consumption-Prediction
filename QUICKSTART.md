# Machine Learning Project: Household Power Consumption Prediction

This is a professional-grade data science project template for predicting household power consumption using machine learning.

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Train the model
python train_pipeline.py

# 3. Start Flask API
python flask_app.py

# 4. Run tests
pytest tests/ -v

# 5. Run with Docker
docker-compose up
```

## Key Features

## Project Structure

- **data/**: Raw and processed datasets
- **src/**: Source code (data, features, models, visualization, utils)
- **tests/**: Comprehensive unit tests
- **notebooks/**: Jupyter notebooks for exploration
- **models/**: Trained model artifacts
